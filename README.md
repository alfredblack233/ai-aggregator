# AI 四开面板 · 豆包/DeepSeek/Gemini/Claude 一体化工具
一个纯前端 HTML 工具，无需安装、开箱即用，一键聚合四大主流 AI 助手，解决多模型使用的效率痛点。

<img width="1586" height="893" alt="image" src="https://github.com/user-attachments/assets/975f6ddf-bb14-4cb8-bc60-a584ba26441e" />


## 🚀 核心功能
1. **四平台一体化展示**
   内置 iframe 嵌入 豆包、DeepSeek、Gemini、Claude 四大 AI 平台，2×2 网格布局同时展示
2. **一键复制提问**
   输入问题后一键复制到剪贴板，无需反复切换页面、粘贴文本
3. **自由拖拽布局**
   支持横向/纵向拖拽分割线，自由调整每个 AI 面板大小，适配不同使用习惯
4. **极致体验**
   拖拽流畅无卡顿、小屏自适应、防文本选中、界面美观简洁

## 📌 适用场景
1. 开机自动启动，一次性打开所有 AI 平台，自动保持登录状态
2. 同一个问题需要对比多个大模型回答，筛选最优解
3. 日常办公/学习/编程，高频使用多款 AI 助手的用户
4. 追求极简效率，不想重复打开网页、复制粘贴的用户

## 💻 使用说明
1. 网页版：直接从本站下载所有文件放在同一个文件夹中，按照豆包的指引安装Node.js等内容，随后在文件夹中shift+右键打开powershell，输入npm start开始使用
2. 完整版：需由最底下链接下载完整压缩包，按照路径打开使用：ai-aggregator\dist\win-unpacked\AI聚合器.exe，无需安装任何依赖

## 🎯 核心价值（解决的核心问题）
1. **彻底解决开机重复操作痛点**
   无需分别打开四个 AI 网址、手动登录，一个文件一键启动所有平台，大幅节省开机准备时间
2. **彻底解决多模型对比效率问题**
   告别反复切换标签页、复制粘贴问题，同一屏幕查看所有回答，快速对比、筛选最优答案
3. **提升使用体验**
   自定义布局、双主题、独立刷新，让多 AI 协作更流畅、更舒适

## 📝 技术说明
- 纯原生 HTML/CSS/JS 编写，无任何第三方依赖
- 本地双击运行，无需部署服务器
- 支持设置开机自启，进一步提升使用效率

## ⚠️ 注意事项
1. 部分 AI 平台（Gemini/Claude）可能需要科学上网才能访问
2. 首次使用需手动完成各 AI 平台的登录操作，登录状态由浏览器自动保存
3. 若面板加载异常，可尝试刷新单个面板或清空浏览器缓存
4. 仅供学习娱乐，请勿商用
